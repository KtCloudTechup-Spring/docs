# 🌱 Spring 팀 - TechUp Challenger Hub

## 📌 소개
부트캠프 챌린저들을 위한 정보 공유 커뮤니티 플랫폼으로,
실시간 채팅, 게시판, 과정별 정보 분류 기능을 제공합니다.

---

## 📁 레포지토리 구성
- `spring_be` — 백엔드(Spring Boot)
- `spring_fe` — 프론트엔드(Next.js)
- `docs` — 문서 및 보고 체계

---

## 🔧 기술 스택
**Frontend:** Next.js, Tailwind, shadcn/ui  
**Backend:** Spring Boot, JPA, WebSocket  
**Infra:** AWS, Docker, Nginx, Redis  
**DB:** MySQL
